Custom Cipher Playground

This tool lets you experiment with classic ciphers like Caesar and Vigenère!
You can encrypt and decrypt your own messages, see how the ciphers work, and learn about the basics of cryptography.

  • Caesar Cipher: Shifts each letter by a number you choose.
  • Vigenère Cipher: Uses a word as a repeating key to shift letters.
